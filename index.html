xprocspec - XProc testing tool

This is an experimental tool for testing XProc scripts. Its grammar is based on XSpec and is implemented in XProc.

Feel free to have a look at the code and give feedback, but note that it is not yet a complete and 100% working implementation.

If you want to try it out, run src/main/resources/content/xml/xproc/xprocspec.xpl with src/test/xprocspec/test1.xprocspec as input.

xprocspec is itself tested using xprocspec. The latest results are available as HTML or as JUnit XML.

There's also a Maven plugin in the works over at @bertfrees/xproc-maven-plugin which will allow you to do automated testing of your XProc projects!
