xprocspec - XProc testing tool

This is an experimental tool for testing XProc scripts. Its syntax is based on XSpec and is implemented in XProc.

Feel free to have a look at the code and give feedback, but note that it is not yet a complete and 100% working implementation.

The latest version of the specification for the XProcSpec grammar is located here: http://josteinaj.github.com/xprocspec/www/documentation/index.html
